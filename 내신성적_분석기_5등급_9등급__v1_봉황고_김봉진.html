📖 사용 설명서
내신성적 분석기 (5등급→9등급)
➕ 파일 추가하는 방법
  1. 상단 ➕ 파일 추가 버튼을 클릭하면 파일 선택창이 열립니다.
  2. NEIS에서 반드시 XLS data 형식으로 다운받은 파일을 선택합니다.
    • 교과학습발달상황: 학교생활기록부 → 학생부 항목별 조회 → 교과학습발달상황 → 💾 저장 → XLS data
    • 지필고사 원점수: 학급담임/부서장 → 지필평가 조회 → 학급별일람표조회-전체교과목 → 💾 저장 → XLS data
  3. 파일 선택 후 📂 파일 관리 모달에서 목록을 확인합니다.
  4. ✅ 확인 버튼을 클릭하면 바로 성적 분석 화면으로 전환됩니다.
💡 파일을 화면 위로 드래그해서 놓으면 파일 선택창 없이 바로 추가됩니다. 여러 파일을 한번에 드래그할 수 있습니다.
⚠️ 같은 학기 여러 반 파일을 모두 추가한 뒤 확인하면 전체 합산 기준으로 정확한 석차백분율이 계산됩니다.
🔄 기존 파일 삭제 및 교체 방법
  1. 상단 📂 파일 관리 버튼을 클릭합니다.
  2. 삭제할 파일을 체크박스로 선택합니다.
    • 전체 선택 — 모든 파일 한번에 선택
    • 전체 해제 — 선택 전체 해제
    • 파일 행을 클릭하면 개별 선택/해제
  3. 🗑 선택 삭제 버튼으로 체크된 파일을 삭제합니다.
  4. ➕ 파일 추가로 새 파일을 선택한 뒤 ✅ 확인을 클릭합니다.
⚠️ 상담 기록은 학생 이름 기준으로 별도 저장되므로, 파일을 교체해도 유지됩니다.
↕ 과목·등급별 정렬 방법

각 탭의 컬럼 헤더(과목명, 등급, 백분율 등)를 클릭하면 정렬됩니다.

  • 1번 클릭 ① — 오름차순 (낮은 값 → 높은 값)
  • 2번 클릭 ①↓ — 내림차순 (높은 값 → 낮은 값)
  • 3번 클릭 — 정렬 해제
  • 다중 정렬: 여러 컬럼을 순서대로 클릭하면 ①②③ 우선순위로 정렬됩니다.
💡 탭을 전환하면 정렬이 초기화됩니다.
📝 상담 카드 작성 방법
  1. 성적표에서 학생 이름을 클릭하면 상담 기록 창이 열립니다.
  2. 내용을 입력하고 💾 저장 또는 Ctrl+Enter로 저장합니다.
  3. 날짜별로 여러 건을 누적 저장할 수 있습니다.
  4. 각 기록의 🗑 삭제 버튼으로 개별 삭제 가능합니다.
✅ 상담 기록은 파일 초기화 후에도 별도 유지됩니다.
🖨 인쇄 기능 사용 방법

① 개인별 인쇄 (성적 + 상담 기록 선택)

  1. 학생 행 오른쪽 끝 🖨 버튼 클릭
  2. 포함할 상담 기록을 개별 체크 후 인쇄 클릭

② 반별 일괄 인쇄

  • 반 헤더(진한 배경)의 🖨 N반 인쇄 버튼 클릭 → 해당 반 전체 출력

③ 상담 기록만 출력 (학생별)

  • 학생 이름 클릭 → 상담 기록 창 오른쪽 상단 🖨 인쇄 클릭
  • 기록을 선택해 성적 없이 상담 내용만 출력됩니다.

④ 전체 학생 상담 기록 일괄 출력

  • 상단 📋 전체 상담 기록 인쇄 버튼 → 기록 있는 학생 전체 한 번에 출력
💡 인쇄 창이 열리지 않으면 브라우저 주소창의 팝업 허용 버튼을 클릭해 주세요.
✏️ 수동 입력 사용 방법

NEIS 파일 없이 성적을 직접 입력해 9등급 예상 범위를 확인할 수 있습니다.

  1. 반 선택 버튼 중 ✏️ 수동 입력을 클릭합니다. (파일 없이 시작하면 자동으로 수동 입력 모드)
  2. 학생 이름을 입력하고 과목명과 5등급을 입력합니다.
  3. 5등급에 소수 입력 가능 (예: 2.50, 3.20) — 소수점 2자리까지 지원
  4. ➕ 과목 추가로 과목을 추가할 수 있습니다.
  5. 5등급 평균과 9등급 변환 예상 범위가 자동 계산됩니다.
  6. ➕ 학생 추가로 여러 학생을 관리할 수 있습니다.
  7. 학생 이름 입력 후 📝 상담 버튼으로 상담 기록 작성 가능합니다.
  8. 🖨 인쇄 버튼으로 성적 분석 카드를 출력할 수 있습니다.
✏️ 수동 입력 데이터는 자동 저장되며 다음에 열어도 유지됩니다. 9등급은 참고용 예상 범위입니다.
